# CH.32
## I2C 통신을 이용한 lcd 2개 display
#### 문제점 및 해결방안
- lcd의 주소값이 같게 세팅되있을 경우엔 각자 컨트롤이 힘듬
- 둘의 주소값을 다르게 입력하여 각각의 객체를 생성함
- 만약 실제 소자로 할 경우엔 납땜으로 주소 변경
